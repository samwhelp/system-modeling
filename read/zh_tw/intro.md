## 常用視窗操作

| 按鍵組合          | 功能     |
| ----------------- | -------- |
| `Win + q`         | 關閉視窗 |
| `Win + w` | 視窗最大化 |
| `Win + x` | 視窗最小化 |
| `Win + f` | 視窗全螢幕 |
| `Win + d` | 切換顯示桌面 |
| `Win + e`         | 開始移動視窗 |
| `Win + r`         | 開始更改視窗大小 |
| `Win + a` | 聚焦切換到前面一個視窗 |
| `Win + s` | 聚焦切換到後面一個視窗 |


> 一般「關閉視窗」的按鍵綁定是在「`Alt + F4`」。

> 一般「視窗全螢幕」的按鍵綁定是在「`F11`」。

> 一般「切換視窗」的按鍵綁定是在「`Alt + Tab`」。
